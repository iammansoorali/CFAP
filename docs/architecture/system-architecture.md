# CarbonWise System Architecture

## Overview
CarbonWise is a production-grade platform to help users understand, track and reduce carbon emissions.

## Modules
- Authentication
- Carbon Engine
- Recommendation Engine
- Analytics
- Goals
- Habits
- Simulator
- Gamification
- Educational Hub
- AI Advisor

## Tech Stack
- Next.js 15
- Express.js
- PostgreSQL
- Prisma
- TypeScript
